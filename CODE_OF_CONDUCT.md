# Code of Conduct

## Our Pledge

We are committed to making participation in this project respectful and constructive for everyone.

## Expected Behavior

- Be respectful and considerate.
- Give and accept technical feedback professionally.
- Keep discussions focused on the project and its users.
- Credit sources and contributors when adding dataset or code changes.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Discriminatory language or behavior.
- Publishing private information without consent.
- Deliberately submitting misleading data, secrets, or unsafe code.

## Reporting

Report conduct concerns to project maintainers through the repository hosting platform. If the concern involves a security issue, follow `SECURITY.md` instead of opening a public issue.

Maintainers may remove comments, close issues, reject merge requests, or restrict participation when behavior violates this code of conduct.
